
# omft-utils

Utility functions for use with Oracle MFT - Managed File Transfer

## Installation

npm install omft-utils --save

## Usage

See examples folder

function isBinary  

    var outils = require('omft-utils');
    console.log(outils.isBinary('test.bin));
    console.log(outils.isBinary('test.xml));
